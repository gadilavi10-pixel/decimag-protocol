Decision behavior before execution

AI can act. But nothing checks if it should.

DECIMAG evaluates whether an action is justified before execution. These examples show how the protocol responds to high-impact requests under different conditions.

Request
Send payment of $50,000 to external vendor account
ESCALATE
Why
Irreversible financial action without sufficient validated context.
Next step
Escalate to human review.
Request
Grant temporary access to Q3 financial reports
REOPENED
Why
Re-entry condition detected. The decision can now be re-evaluated with updated context.
Next step
Re-check the request with updated context.
Request
Disable user account after suspicious activity alert
ESCALATE
Why
High-impact account action requires verified justification before execution.
Next step
Escalate to human review.
What this shows
ESCALATE for irreversible or high-impact actions.
REOPENED when valid re-entry conditions exist.
ALLOW only when the action meets all authorization conditions.
DECIMAG determines whether an action is justified before it executes.
