Why this matters

This is not post-event control.

Not monitoring

Monitoring tells you what already happened. DECIMAG decides whether it gets to happen.

Not output filtering

This is not about cleaning model text. It is about authorizing or withholding real-world action.

Rules break here

Rules don’t fail because they’re wrong. They fail because they don’t see the situation.

Core principle

In autonomous execution, capability is not enough.

DECIMAG inserts a decision authorization protocol between system intent and execution, so authority is granted by design, not assumed by default.
